#### oo基础
1. 抽象
2. 封装
3. 多态
4. 继承
#### oo原则
1. 封装变化
2. 多用组合，少用继承
3. 针对接口编程，不针对实现编程
#### oo模式
> 策略模式:
定义了算法族，分别封装起来，让它们之间可以互相替换，此模式让算法的变化独立于使用算法的客户。
